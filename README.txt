Bienvenue, ce dossier contient:

==> un dossier lib, dans lequel il y a deux fichiers:

          > un nommé player.rb, qui rassemble toutes les classes et méthodes  nécessaires au lancement des programmes de jeu.

          > un nommé game.rb, qui est malheureusement vide.

==> un fichier Gemfile, qui renseigne sur la version de ruby utilisée ainsi que les gem nécessaires au bon lancement des programmes.

==> deux fichiers: 
          > un nommé app.rb, qui appellera les méthodes présentes dans les fichiers du dossier lib afin que le premier programme se lance sans erreur.
          > un  nommé app2.rb, qui appellera les méthodes présentes dans les fichiers du dossier lib afin que le deuxième programme se lance sans erreur.
